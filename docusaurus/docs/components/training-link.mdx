---
title: Training Link
summary: Component for allowing link out to training in the Header component
---

Provide a link for training videos for your app

## Installation

```bash
npm install @availity/training-link --save
```

## Example

```jsx  viewCode=true
import TrainingLink from '@availity/training-link';

<TrainingLink link="https://www.github.com/availity" name="Availity Github" />;
```

## Props

### `link: string`

Link to training video.

### `name: string`

Name of your app to make the sentence complete.
