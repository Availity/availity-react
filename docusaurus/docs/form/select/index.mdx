title: Select
---

Wrapper for react-select-async-paginate to work with formik.

### Installation

```bash
npx install-peerdeps @availity/select --save
```

### Validation

See [yup](https://github.com/jquense/yup) and [@availity/yup](https://github.com/Availity/sdk-js/tree/master/packages/yup)
