# Deprecated Packages

The packages in this section are deprecated and no longer actively maintained. Future updates are not guaranteed.

These packages depend on Bootstrap, reactstrap, or availity-uikit for styling. Users should migrate to [`@availity/element`](https://availity.github.io/element), which is the modern replacement built on a new design system.

## Not Compatible with React 19

The following packages cannot be upgraded to React 19 due to the APIs they use:

- `@availity/reactstrap-validation-date`
- `@availity/reactstrap-validation-select`
- `@availity/upload`
