Check user permissions to see if the current user is authorized to see your content. This packages exports a [component](./authorize) and [hook](./useAuthorize) for validating authorization

## Installation

npm

```bash
npm install @availity/authorize @availity/api-axios axios
```

Yarn

```bash
yarn add @availity/authorize @availity/api-axios axios
```
