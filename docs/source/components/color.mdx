Colors are used throuhgout the Availity portal across various components.  
Color impact users' emotions, draws their attention, and put them in the 
right frame of mind to take action. It's also one of the main factors in 
customers' perception of a brand.


## Implementation 

### Installation

These colors come in natively with Availity reactstrap components.

##  Accessibility

Availity is working towards creating a 508 compliant portal experience. This is an extensive undertaking that takes time. 
If at any time you find or question a component in this library to be out of 508 compliance please notify a member of the
User experience team.
